# Snake-Water-Gun-game
A simple Python-based command-line game inspired by the classic “Rock Paper Scissors” concept. The player competes against the computer by choosing between Snake, Water, or Gun, where each choice interacts with the others based on predefined rules.

🎮 How it works
Snake drinks Water → Snake wins 🐍
Gun shoots Snake → Gun wins 🔫
Water drowns Gun → Water wins 💧
Same choice → Draw 🤝
The computer randomly selects its move, and the result is decided instantly after each round.
✨ Features
🤖 Computer vs Player mode
🎲 Randomized computer choice
🔁 Multiple rounds gameplay
⚡ Instant result display
🧠 Logic-based decision system using conditionals
🛠 Tech Stack
Python (core logic)
Random module (for computer moves)
🚀 Purpose
A beginner-friendly project to practice conditional statements, loops, and basic game logic while building something fun and interactive.

🏆Score rules
user wins: user gets +1
computer wins: computer gets +1
draw: both get +1
